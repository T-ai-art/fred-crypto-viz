# CryptoExplorer — iPhoneホーム画面に追加する手順

## 手順

1. iPhoneの **Safari** で以下のURLを開く（Chrome等では不可）

   https://t-ai-art.github.io/fred-crypto-viz/

2. 画面下部の **共有ボタン**（□に↑のアイコン）をタップ

3. メニューを上にスクロールし、**「ホーム画面に追加」** をタップ

4. アプリ名（デフォルト: CryptoExplorer）を確認して **「追加」** をタップ

5. ホーム画面に金色のチャートアイコンが追加される。タップするとフルスクリーンで起動。

## 補足

- Safari以外のブラウザ（Chrome, Firefox等）では「ホーム画面に追加」が表示されない。必ずSafariを使用すること。
- PWA（Progressive Web App）として動作するため、一度読み込めばオフラインでも直前のキャッシュデータで閲覧可能。
- データは1時間ごとに自動リフレッシュされる。手動で更新したい場合は青い「Refresh」ボタンをタップ。
- CSV/Excelダウンロードにはパスワードが必要。
